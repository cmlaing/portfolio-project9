Wheel of Success



This fun, interactive word guessing game features JavaScript at its core. The user clicks the onscreen keyboard to choose a letter. If the letter is in the unknown phrase, it will light up and the chosen key becomes disabled. If the letter is not in the phrase, the user loses a heart icon. The game ends with a win if the phrase is guessed before running out of heart icons, or it ends with a loss if the phrase was not correctly guessed before the user ran out of heart icons. What do you win as a prize? Some might say it is of the cute and cuddly variety... Play to find out more.

Web Style Guide



Are you a fan of Sass? This well organized project demonstrates the power of Sass! As a companion to Cascading Style Sheets (CSS), Sass helps implement these styles in a modular format. Although this project is on the smaller side, a larger project with multiple team members would benefit from such a modular approach like Sass. Each section has been broken down into three subcategories: base, components, and utilities. The Base section includes things like typography and normalizing style sheets for a level playing field. The Utilities includes variables and mixins - repeatable code to "mix into" your regular style sheets. Finally, the Components section includes bits and pieces that make up your website like buttons, form, the overall grid, navigation, and media queries. It may sound like a bit to take in, but it is actually quite straight-forward and a very useful tool to be familiar with.
